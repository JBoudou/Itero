Display the poll in the list Spread the link

The title of the poll should be the question the poll will answer to.

The optional description may give some additional information, for instance about the different alternatives.

The title will be displayed in the list of polls, allowing any user to participate in. If you don't want the poll to be public, select "Spread the link" as Publicity.

The title will be displayed in the list of poll only for the participants in the poll. You will have to send the link of the to all the participants. If you prefer your poll to be public, select "Display the poll in the list" as Publicity.
